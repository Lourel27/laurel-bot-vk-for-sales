# laurel-bot-vk-for-sales
Бот для ВКонтакте для продаж сообщества. Дает информацию по горячим кнопкам ВК и зовет администратора со звуковым сопровождением. Администратору показывается окно с кнопкой "Ответил". Запись лога между нажатием кнопки и появления сообщения в логах по ...\LAURELL_VK_BOT\dist\admin_calls.log
Бот создан под виртуальным окружением .venv, библиотеки все внутри.

Заменить звук проигрываемый при вызове бота - файл callVK.wav
Расширение ТОЛЬКО wav. Название callVK, объем и длина аудио неважна.

Все библиотеки хранятся .venv\Lib\site-packages

Экзешник создается в папке dist. 

Каждая строка подписана в коде. Сделано все для удобства.

создать exe через батник EXE CREATE
